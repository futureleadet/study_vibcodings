# **'뉴트리 가이드' 단계별 기능 로드맵 (Feature Roadmap)**

| 기능 분류 | Phase 1: MVP (핵심 가설 검증) | Phase 2: Next Step (기능 고도화) | Phase 3: Future Scope (비전 확장) |
| :---- | :---- | :---- | :---- |
| **G-01: 회원 및 프로필** | \- 소셜 로그인 (카카오/네이버) \- 필수 프로필 입력 (닉네임, 키, 현 체중, 목표 체중) | \- 일반 이메일 회원가입/로그인 \- 상세 프로필 (알레르기, 선호 음식, 운동량) | \- 프로필 연동 (건강검진 데이터 등) |
| **G-02: 식단 기록 (Feature 1\)** | \- **AI 사진 인식 (보조):** 음식명 '제안' (유저 확인/수정) \- 음식명 DB 기반 칼로리 자동 계산 \- 수동 입력 (검색 및 직접 입력) | \- **AI 피드백:** "칼로리가 높아요", "단백질이 부족해요" 등 간단한 코칭 \- 영양소(탄/단/지) 분석 | \- **AI 자동 인식 (완전 자동):** 100% 자동 분석 \- 영상 기반 식단 분석 \- 레시피 및 재료 기반 상세 분석 |
| **G-03: SNS 및 공유 (Feature 2\)** | \- '내 기록' 이미지 카드로 생성 (인스타그램 공유) \- 앱 내 '친구 피드' (팔로우 기반) \- '좋아요' 및 간단한 이모티콘 피드백 | \- 앱 내 상세 댓글 기능 \- **'전문가' 피드백 기능 (BM 핵심)** \- 그룹 챌린지 기능 (예: 1주 챌린지) | \- 타 SNS(트위터, 페이스북) 연동 \- 위치(LBS) 기반 맛집/식단 공유 |
| **G-04: 동기부여 (Feature 4\)** | \- **칭찬 뱃지:** 특정 목표 달성 시 지급 (예: 3일 연속 기록) \- 'My 뱃지' 수집 페이지 | \- 랭킹 시스템 (친구 간 / 전체) \- 포인트 및 리워드 시스템 (기프티콘 연동) | \- 오프라인 연계 (피트니스 센터 할인 등) |
| **G-05: 히스토리 분석 (Feature 5\)** | \- **대시보드:** 오늘 섭취/권장 칼로리 (도넛 그래프) \- 최근 7일 섭취 칼로리 (라인 그래프) \- '기록' 탭 (캘린더 뷰, 리스트 뷰) | \- 주간/월간 상세 리포트 \- 시간대별 섭취 패턴 분석 | \- 상세 건강 리포트 (PDF 다운로드) \- 외부 데이터(수면, 운동) 통합 분석 |
| **G-B: 기타  (Feature 3, 6\)** | \- **개인화 홈 (간단):** 유저 닉네임, 뱃지 현황 노출 | \- **경고/알림 기능 (Feature 3):** 목표 칼로리 초과, 기록 시간 알림 (Push) | \- **아바타 예측 (Feature 6):** 현재 데이터 기반 목표 달성 시 변화 예측 |
